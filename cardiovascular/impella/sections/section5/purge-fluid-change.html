Section 5: Changing the Purge Fluid

Version 3.2 | March 2025 – December 2026

ICU nurses are expected to change the purge fluid. If required, changing the purge cassette is the responsibility of the cath lab physiologist team.

Procedure

  1. Prepare a new bag of purge solution.

    Standard is:

    500mls 5% glucose
    +
    12.5mls 8.4% sodium bicarbonate

  2. Select "Purge Menu" using soft buttons

    Step 2: Select Purge Menu on AIC

  3. Select "Change purge fluid" using dial
  4. Select "OK" to bolus purge fluid using dial
  5. Change purge fluid

    Step 5: Purge fluid change procedure

  6. Wait for the automatic purge purge cycle to stabilise

    The system will automatically resume normal purge pressure (300-1100mmHg)

  7. Document the purge fluid change in the patient's notes
  8. Ensure old bag is disposed of appropriately
