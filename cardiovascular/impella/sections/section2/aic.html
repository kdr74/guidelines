Automated Impella Controller (AIC)

The Automated Impella Controller (AIC) is the primary user control interface for the Impella catheters. It provides automated monitoring, alarms, performance control, and information on catheter position.

Figure 3: Impella Console - Automated Impella Controller (AIC)

Key Functions

  • Automated alarms for position, flow, and purge system issues
  • Performance control via adjustable P-levels (P1-P9)
  • Catheter position monitoring using optical sensors and waveform analysis
  • Priming and de-airing capabilities
  • Automatic purge system that adjusts pressure to maintain constant flow through the device, preventing thrombosis of the micro-axial pump

Operating Modes

AUTO Mode (Recommended)

The Impella CP with SmartAssist will normally be run on "AUTO" mode, where:

  • The desired P-level is set by the clinical team
  • The AIC automatically adjusts the P-level to derive the best flow rate
  • Minimizes suction events and pressure alarms
  • Optimizes device performance based on patient haemodynamics

Manual Mode

Manual mode maintains the P-level selected by the clinician without automatic adjustment. This mode:

  • Has an increased risk of suction alarms
  • Is useful for evaluating effects of preload and afterload changes on flow rates
  • Required during:
    • Cardiac arrest (reduce to P2 until ROSC achieved)
    • Catheter repositioning (reduce to P2)

P-Levels and Flow Rates

The AIC provides 9 performance levels (P1-P9) that control the speed of the micro-axial pump and therefore the flow rate delivered by the device.

Figure 4: P (Performance) levels and likely flow rates with correct catheter position and loading conditions

Key points about P-levels:

  • P1-P3: Lower flow rates (0.5-2.0 L/min) - white LV placement signal waveform not displayed
  • P4-P9: Higher flow rates (2.5-4.3 L/min) - all three waveforms visible
  • Actual flow achieved depends on catheter position and patient loading conditions
  • In AUTO mode, the AIC may adjust between adjacent P-levels to optimize flow

Console Displays

The AIC provides several screen displays including:

  • Placement Screen - Shows pressure waveforms and motor current (see Placement Screen & Waveforms section)
  • Trends Screen - Historical data on flow, pressures, and alarms
  • Alarms Screen - Current and historical alarm information
  • Purge Screen - Purge system parameters and settings

Data Recording

The AIC automatically records:

  • Continuous waveform data
  • Flow rates and P-level settings
  • Alarm events with timestamps
  • Purge system parameters

This data can be downloaded for review and quality assurance purposes.
