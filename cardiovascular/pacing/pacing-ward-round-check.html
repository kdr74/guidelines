Clinical Decision Flowchart

Clinical decision flowchart for daily temporary pacing checks

Step-by-Step Process

Initial Safety Assessment

  • Determine pacing dependency: Is the patient fully pacing-dependent?
  • If YES: Confirm consultant presence on unit before proceeding
  • If NO: Proceed with routine checks

Step 1: Assess Underlying Rhythm

  • Slowly decrease the pacing rate below the patient's intrinsic rate
  • Observe for stable underlying rhythm
  • If stable rhythm present: Consider obtaining 12-lead ECG and document
  • If unstable or no rhythm: Document as pacing-dependent and skip to Step 2

Step 2: Review Pacing Settings

  • Confirm current mode is appropriate (VVI vs DDD)
  • Verify rate setting is optimal for patient
  • Reassess ongoing indication for pacing
  • Make adjustments if needed and document changes

Step 3: Check Output Threshold

  • Set pacing rate above patient's intrinsic rate
  • Gradually decrease output until loss of capture
  • Record the threshold value (minimum output for capture)
  • Set final output at 2√ó threshold (typically 25-50% above threshold)
  • If no capture at maximum: Seek senior help immediately
  • If threshold >10V: Escalate to Cardiology

Step 4: Check Sensitivity

  • Press PAUSE button on pulse generator
  • Observe P wave and R wave amplitudes
  • If R wave visible: Set V-sense to 1/3 of R wave amplitude (or as low as possible, whichever is higher)
  • If no rhythm visible: Set V-sense to default 1mV
  • Document sensitivity setting in clinical notes

Critical Stop Points

‚ö†Ô∏è Mandatory Stop: Pacing-Dependent Patient

If patient is pacing-dependent and Consultant is NOT on unit:

  • STOP the ward round check immediately
  • Wait for Consultant to arrive before proceeding
  • Do not attempt threshold checks without consultant supervision
  • Reason: Loss of pacing during checks could be life-threatening

üö® Escalation Required: High Threshold

If threshold >10V:

  • Escalate to Cardiology immediately
  • Consider alternative pacing methods
  • Increase output to maximum if needed for patient safety
  • High threshold suggests wire dislodgement or exit block

üö® Emergency: Loss of Capture

If unable to achieve capture even at maximum output:

  • Seek senior help immediately
  • Prepare for alternative pacing (transcutaneous if needed)
  • Check all connections and battery status
  • Consider wire dislodgement or failure

Final Documentation

Document in Clinical Information System (CIS):
  • Presence and characteristics of underlying rhythm
  • Threshold value recorded during check
  • All current pacemaker settings (mode, rate, output, sensitivity)
  • Plan for ongoing pacing vs weaning vs permanent pacemaker
  • Any complications or concerns identified
  • Consultant notification if escalation required

Decision Outcomes

Continue Temporary Pacing

If ongoing need for pacing identified:

  • Continue current pacing strategy
  • Review daily during ward rounds
  • Reassess underlying rhythm regularly

Plan for Weaning

If stable underlying rhythm with good rate:

  • Consider reducing backup pacing rate
  • Plan for wire removal once criteria met
  • Ensure patient afebrile >24 hours before removal

Consider Permanent Pacemaker

If persistent pacing requirement:

  • Discuss with Cardiology/Consultant
  • Ensure patient afebrile >24 hours before PPM insertion
  • Document indication for permanent pacing
Remember: Patient safety is paramount. When in doubt, consult with a senior colleague before proceeding with threshold checks or making significant changes to pacing settings.
